# MySQL原理分类

本文对MySQL的核心原理进行分类介绍。

<!-- more -->

## 1. 基础架构

MySQL采用客户端/服务器架构,主要包含:

- 连接层 - 处理客户端连接
- 服务层 - SQL解析优化执行
- 存储引擎层 - 数据存储和读写

## 2. 事务机制

事务是MySQL的重要特性,包括:

- ACID特性
- 隔离级别
- MVCC多版本并发控制
- 锁机制

## 3. 日志系统 

MySQL的主要日志包括:

- binlog - 二进制日志
- redo log - 重做日志  
- undo log - 回滚日志

## 4. 索引体系

索引是优化查询的关键:

- B+树索引
- 哈希索引
- 全文索引
- 空间索引

## 5. 优化机制

查询优化的核心机制:

- 查询缓存
- 执行计划优化
- 统计信息收集
- 成本估算
